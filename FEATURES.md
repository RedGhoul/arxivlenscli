# Feature Roadmap

Organized by implementation difficulty and impact.

## 🟢 Easiest Wins (Quick to implement, high value)

1. **Copy Link** - Copy arXiv URL to clipboard
2. **Theme Switching** - Toggle dark/light modes
3. **Quick Search** - Global shortcut to search from anywhere
4. **Reading History** - Track recently viewed papers
5. **Keyboard Customization** - Configure keybindings in settings

## 🟡 Medium Effort / High Impact

6. **Bookmark/Favorites** - Save papers with tags
7. **Search History** - Save & rerun previous searches
8. **Related Papers** - API endpoint already exists
9. **Advanced Filters** - Extend existing search params
10. **Export/Import Bookmarks** - JSON/CSV for backup
11. **Notifications** - New papers in subscribed categories/authors
12. **Paper References** - View references from paper details
13. **Statistics Dashboard** - Reading stats, favorites count

## 🔴 Complex / High Impact

14. **Citation Network** - Papers that cite/refer to current paper
15. **AI Recommendations** - Personalized suggestions based on reading
16. **Topic Trends** - Trending topics in arXiv
17. **Download Manager** - Batch download PDFs
18. **Paper Comparison** - Side-by-side view of two papers
19. **Paper Annotations** - Add notes to papers
20. **Author Collaboration Network** - Co-author relationships
21. **Offline Mode** - Cache papers locally
